## ❓ 提问

### 1: 为什么没有想要的日志内容

日志内容需要手动写入, 详细示例demo中几乎都有体现, 请自行查看

### 2: 为什么没有测试报告

- html：自动生成，检查 run 参数是否开启 html 报告创建, 默认开启

- yaml：自动生成，它目前仅用于测试结果统计

- allure：自动生成，默认开启并自动打开浏览器访问, 前提已正确安装 allure 程序
